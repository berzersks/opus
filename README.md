# PHP Opus Extension

Uma extensão PHP para codificação e decodificação de áudio usando o codec Opus, fornecendo uma interface simples para trabalhar com áudio comprimido de alta qualidade.

## Sobre o Opus

Opus é um codec de áudio totalmente aberto, livre de royalties, altamente versátil. É padronizado pela Internet Engineering Task Force (IETF) como RFC 6716. Foi desenvolvido pela Xiph.Org Foundation e é amplamente usado para transmissão de áudio em tempo real, streaming e armazenamento de áudio.

## Características

- ✅ Codificação de áudio PCM para formato Opus
- ✅ Decodificação de áudio Opus para PCM
- ✅ Suporte para diferentes taxas de amostragem
- ✅ Suporte para mono e estéreo
- ✅ Interface PHP orientada a objetos
- ✅ Otimizado para aplicações VoIP

## Dependências

- PHP 7.4+ ou PHP 8.x
- libopus-dev (biblioteca Opus)
- Ferramentas de desenvolvimento PHP (php-dev)
